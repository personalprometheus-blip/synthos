Getting Started
Complete these steps to activate your trading agent
1
Set Up Two-Factor Authentication
Protect your account with an authenticator app. We recommend Microsoft Authenticator — it works with Synthos and your Alpaca trading account.
 iPhone (App Store) ▶ Android (Google Play)
After installing, open the app and add your Alpaca account using the QR code from Alpaca's security settings.
2
Create Your Alpaca Trading Account
Alpaca is the brokerage that executes your trades. Synthos connects to Alpaca through their API — a secure key that gives the trading agent permission to place orders on your behalf. You maintain full control of your Alpaca account at all times.
Visit alpaca.markets →
Sign up for a free account. Start with Paper Trading (simulated) to test the system before using real money.
3
Find Your Alpaca API Keys
API keys are like a password that lets Synthos communicate with your Alpaca account. You need two keys: an API Key and a Secret Key.
How to find your keys:
  1. Log in to your Alpaca account at app.alpaca.markets
  2. Click your profile icon (top right) → select API Keys
  3. Click Generate New Key (or Regenerate if you already have one)
  4. Copy both the API Key ID (starts with PK...) and the Secret Key
  5. Important: The Secret Key is only shown once. Save it securely.
Make sure you are generating keys for Paper Trading (not Live) until you are ready to trade with real money.
4
Connect Your Keys to Synthos
Paste your API keys into the Settings page. Synthos encrypts and stores them securely — they never leave this system.
  1. Go to Settings in the sidebar
  2. Find the API Keys section
  3. Paste your Alpaca API Key and click Update
  4. Paste your Alpaca Secret Key and click Update
  5. Click the Test button to verify the connection
5
Choose Your Trading Style
Open the Agent Configuration panel (gear icon on the right edge of the screen) and select a preset: Conservative, Moderate, or Aggressive. You can customize individual parameters later.
The agent starts in Paper Trading mode by default. It will simulate trades using real market data without risking real money. Switch to Live only when you are confident in the system.

# SYNTHOS RETAIL — updated 2026-04-20
# All times America/New_York (Pi5 16GB system clock)
# Market daemon (intel): news, sentiment, macro, validator, screener, price poller (9:10-16:00)
# Trade daemon (action): continuous trader during market hours 9:30-16:00
# Cron handles: overnight news, boot, heartbeat, backup, maintenance

# Boot services

# ── MARKET DAEMON + TRADE DAEMON — now managed by systemd, not cron ──
# See synthos_build/ops/systemd/pi5/ for the unit files. Timer-driven:
#   synthos-market-daemon.timer          fires Mon-Fri 09:10 ET
#   synthos-trade-daemon.timer           fires Mon-Fri 09:25 ET
#   synthos-sector-backfill.timer        fires *-*-* 02:15 ET (FMP sector lookups)
#   synthos-ticker-identity-agent.timer  fires *-*-* 02:30 ET (nightly identity sweep)
#   synthos-ticker-state-auditor.timer   fires *-*-* 02:35 ET (gap report)
# systemd handles crash recovery (Restart=on-failure) and boot-time
# catch-up (Persistent=true). No cron entry needed for these.
#
# Install on pi5:
#   sudo cp synthos_build/ops/systemd/pi5/synthos-*.{service,timer} /etc/systemd/system/
#   sudo systemctl daemon-reload
#   sudo systemctl enable --now synthos-market-daemon.timer synthos-trade-daemon.timer \
#                               synthos-sector-backfill.timer \
#                               synthos-ticker-identity-agent.timer synthos-ticker-state-auditor.timer
